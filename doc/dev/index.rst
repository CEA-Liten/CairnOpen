.. admonition:: What is |persee|?

	- |persee| is an optimizer tool for energy system studies. It allows to perform techno-economic and environmental analysis of energy systems 
	  by optimization of both the sizing of power systems and the operation planning.
	- |persee| is based on the Mixed Integer Linear Progamming formalism (|milp|).
	- |persee| has been build to allow for several purposes :
	
		- allow to build any architecture of multi-energy system with a verified library of components.
		- allow non-specialists to build easily optimisation problems, and analyze the results.
